# cdot Paper — Working Directory

Target journal: **Bioinformatics (Oxford)**
Article type: Original Paper (≤7 pages / ~5,000 words) — or Application Note (≤4 pages) if content fits

## Files

| File | Purpose |
|------|---------|
| `bioinformatics_submission_guide.md` | Journal requirements summary |
| `abstract.md` | Structured abstract (Motivation / Results / Availability / Contact) |
| `introduction.md` | Background, problem statement, cdot overview |
| `methods.md` | Data sources, JSON format, generation pipeline, access methods |
| `results.md` | Benchmarks: transcript coverage, speed, ClinVar accuracy, T2T, MANE |
| `discussion.md` | Positioning, limitations, future directions |
| `references.md` | Formatted reference list |
| `figures.md` | Figure descriptions and drafts |
| `supplementary.md` | Supplementary tables and figures |
| `notes.md` | Working notes, decisions, things to resolve |

## Key resources

- `claude/paper_thoughts.md` — paper framing, scope decisions, pre-submission checklist
- `literature/literature_review.md` — literature synthesis with all references
- `claude/research.md` — deep technical notes on the codebase
- `claude/open_issues.md` — open GitHub issues relevant to paper features
